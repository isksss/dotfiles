# create-branch

このスキルは、Gitリポジトリ内で新しいブランチを作成するために使用される。ユーザーがブランチ名を指定すると、その名前で新しいブランチが作成される。ブランチは、現在の作業ブランチから分岐して作成される。

## 手順

1. 現在の作業内容を簡単にまとめる。
2. まとめたものから2,3語程度のブランチ名を作成する。
3. `git checkout -b {ブランチ名}` コマンドを実行

## ブランチ名

`{gitflow}/{要約内容}_{yyyymmdd}`の形式でブランチ名を作成する。

- `{gitflow}` は、ブランチの種類を表すプレフィックスで、以下のいずれかを使用する。
  - `feature`: 新機能の開発
  - `bugfix`: バグ修正
  - `hotfix`: 緊急のバグ修正
  - `release`: リリース準備
- `{要約内容}` は、現在の作業内容を簡潔に表す2,3語程度の要約。
- `{yyyymmdd}` は、ブランチを作成した日付を表す。
