# isksss's dotfiles

## requirements
- git
- zsh
- make

## Installation
```sh
# install
$ curl -L https://raw.githubusercontent.com/isksss/dotfiles/main/install.sh | sh
$ cd ~/dotfiles
$ make install

# or

$ git clone git@github.com:isksss/dotfiles.git ~/dotfiles
$ cd ~/dotfiles
$ make install
```
