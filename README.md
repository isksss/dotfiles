## commands memo
### pyenv
1. `pyenv install --list`  
これでインストールできるバージョンを確認  
2. `pyenv install $version`  
$versionのpythonをインストール  
3. `pyenv $scope $version`
$scopeで使用する$versionを指定する。  
$scope = [shell, local, global]
