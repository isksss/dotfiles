# dotfiles

## セットアップ
1. `setup.sh`を実行

## ディレクトリ説明
| ディレクトリ名 | 説明                                   |
| :------------: | :------------------------------------- |
|    .config     | XDG_CONFIG_HOMEに入るファイル群        |
|      home      | ホームディレクトリに配置するファイル群 |
|     script     | スクリプト                             |

## オプション
| オプション | 説明                                 |
| :--------: | :----------------------------------- |
| -o <os名>  | OSごとのスクリプトを実行する(未実装) |
|     -r     | Rustをインストール                   |
|     -p     | Ryeをインストールする                |
|     -c     | SHELLをZSHに変更する                 |
