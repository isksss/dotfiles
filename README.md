# dotfiles

## セットアップ
1. 必要に応じて`script/common.sh`を実行
2. OS別に`script/<OS名>.sh`を実行
3. `setup.sh`を実行

## ディレクトリ説明
| ディレクトリ名 | 説明                                   |
| :------------: | :------------------------------------- |
|    .config     | XDG_CONFIG_HOMEに入るファイル群        |
|      home      | ホームディレクトリに配置するファイル軍 |
|     script     | スクリプト                 |
