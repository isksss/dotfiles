### commit massage
下記のルールに沿ってコミットメッセージを書いていく。  
| long | short | memo |
| :--: | :---: | :--: |
| add: | a: | ファイルやフォルダを追加したとき |
| update: | u: | 既存のファイルに新機能追加したときや、改修したとき |
|fix: | f: | バグ修正やタイポなど |
|wip: | w: | 作業途中など、バックアップや記録のためにコミットしたとき |
